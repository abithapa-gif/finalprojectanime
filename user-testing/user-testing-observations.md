# User Testing Observations — Anime Vault

---

## Tester A — Casual anime fan (phone, moderate tech)

**Task 1: Find Attack on Titan info**
- Tapped "Series" in the nav right away
- Scrolled down the grid looking for AOT — found it, read the card
- Said "87 episodes — okay that's actually doable" unprompted (this validated the decision to show episode counts)
- Took a moment to find the studio info because it was below the description
- Suggestion: "Can you put like the most important stuff first on the card?"

**Task 2: Send a contact message about fan theories**
- Found Contact in the nav, opened the form
- Filled first name, last name, email no problem
- Paused at "Subject" dropdown, read all options, selected "Fan Theory or Discussion"
- Filled in favourite series: "One Piece"
- Started typing in the message box, then stopped and asked "does this actually send?"
- Said the dark form felt "legit, like a real website"

**Task 3: Find who runs Anime Vault**
- Went to About, scrolled past the intro section
- Found the team cards, read each one
- Said "these are funny" (about the team bios)
- Found the info easily

## Tester B — Non-anime watcher (desktop, lower tech)

**Task 1: Find Attack on Titan info**
- Hovered over "Series" nav link, clicked it
- Was overwhelmed briefly by the grid: "There's a lot going on here"
- Found AOT after scanning all 5 cards
- Read the info correctly, noted "I didn't know this show was finished"
- Didn't interact with the filter buttons at all

**Task 2: Contact form**
- Found the form
- Was confused by "Fan Username (optional)" — asked "what's a fan username?"
- Completed the form otherwise without issues
- Didn't notice the success message block below the form
- Said the form felt "professional"

**Task 3: Find who runs the site**
- Clicked About immediately
- Skimmed the intro text (didn't read it all)
- Found the team section, identified 3 people
- Said "I don't know who these people are but that's fine"

## Tester C — Hardcore anime fan + web dev student (desktop, high tech)

**Task 1: Find Attack on Titan info**
- Found AOT in about 4 seconds
- Said "nice that you have the studio listed"
- Noticed: "The filter buttons don't actually work right?" — I explained CSS-only, they understood
- Said the filter buttons gave no feedback when clicked: "They should at least change state visually"

**Task 2: Contact form**
- Completed it very quickly
- Checked the form at different viewport widths using browser dev tools
- Said "the two-column name layout breaks gracefully on mobile, good"
- Noticed the success message wasn't connected to the submit button: "You should at least note that in comments for the instructor"

**Task 3: Find team info**
- Found it, read the bios
- Said "Yuki's Erza Scarlet take is correct"
- Suggested adding some anime-specific flavor to the mission statement
